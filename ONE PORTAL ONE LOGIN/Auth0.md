2.what is "app_type": "regular_web" in auth0 explain it
	In **Auth0**, the `"app_type": "regular_web"` refers to the application type defined when creating or configuring a client application within the Auth0 dashboard. This specifies the type of application being integrated with Auth0's authentication and authorization services.
3.What is a First-Party Application?
	A **first-party application** is an application that is developed and owned by the same entity managing the Auth0 tenant. In simpler terms, it is a trusted application that belongs to the organization configuring the Auth0 system.
